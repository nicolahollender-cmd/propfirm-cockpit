NEXTGEN INVESTMENT GROUP License Key Generator

Pro license key generator

Generates checksum-valid license keys for the Evaluation Cockpit's Pro unlock. Each key is 16 characters (format XXXX-XXXX-XXXX-XXXX) where the final character is a checksum of the first 15 — the cockpit validates this entirely in the browser, with no server or third-party API involved.

How to use this

Generate a batch of keys (e.g. a few hundred) and download the CSV. Each row is one unused key. In your GoHighLevel (or other) automation, pull the next unused key from this list and send it to the customer after a Stripe purchase — e.g. import the CSV as a contact custom value pool, or paste keys one-per-row into a Google Sheet your workflow reads from. Mark each key used as you go (a simple spreadsheet column works fine) so the same key isn't sent twice. Since validation happens locally in the cockpit, no key ever needs to be "registered" anywhere for the unlock to work — the checksum alone is what the cockpit checks.
